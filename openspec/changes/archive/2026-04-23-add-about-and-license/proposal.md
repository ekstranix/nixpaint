## Why

The project has no visible attribution, license, or links to the author/source. Users should be able to find out who made it and where the code lives, and the repo needs a proper MIT license file.

## What Changes

- Add an "About" button to the toolbar that opens a modal/popover overlay
- The overlay shows: project name, copyright (Pim Snel), MIT license mention, link to extranix.com, link to the GitHub repo
- Add a `LICENSE` file (MIT) to the repository root

## Capabilities

### New Capabilities
- `about-dialog`: Toolbar button and modal overlay showing project info, attribution, and links

### Modified Capabilities

_None._

## Impact

- `src/components/Toolbar.tsx`: New About button
- New component for the about modal/overlay
- New `LICENSE` file at repo root
